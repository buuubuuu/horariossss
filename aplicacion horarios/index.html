Manual de usuario

Al abrir la aplicación encontrara un menú con las siguientes opciones:

•Instrucciones

•Horarios

Si presiona en instrucciones, usted vera una página con dos botones para escoger el idioma en el que quiere leer el manual de usuario.

Si presiona en horarios, usted vera una página con los nombres de los maestros, si presiona en el nombre de algún maestro usted vera su horario.

En cada página de la aplicación, hay un botón para regresar al menú principal y otro botón para volver a la página anterior.

User manual

When you open the application you will find a menu with the following options:

•Instructions

•Schedules

If you press on instructions, you will see a page with two buttons to choose the language in which you want to read the user manual.

If you click on schedules, you will see a page with the names of teachers, if you click on the name of a teacher you will see their schedule.

On each page of the application, there is a button to return to the main menu and another button to return to the previous page.
